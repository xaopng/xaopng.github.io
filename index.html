文章中博客根目录均使用[BlogRoot]代替 Hexo 文章生成永久链接 Hexo 文章链接默认的生成规则是：:year/:month/:day/:title，是按照年、月、日、标题来生成的。 通过这种方式生成的链接非常长，如果我们的 Markdown 使用中文标题，URL 一转码，将是一场灾难 更难受的是如果我们修改了文章的日期或者标题，那么将导致链接改变，别人或者你分享出去的文章就会 404 所以就有了这种插件，不论你如何修改文章的日期、名称，只要不改变 footer-matter 中的 id 值，那么文章链接永远不会变 安装插件hexo-abbrlink 1npm install hexo-abbrlink --save 修改配置文件_config.yml 修改文章链接格式 12- permalink: :year/:month/:day/:title/+ permalink: posts/:abbrlink.html 在_config.yml文件末尾添加下面配置 1234567891011121314## abbrlink ...
